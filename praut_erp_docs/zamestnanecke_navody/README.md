# Zaměstnanecké návody pro Praut Huly

Tato složka je určena pro běžné používání Praut prostředí v Huly. Neřeší import, instalaci ani technické založení workspace. Tyto návody popisují, jak má tým pracovat s firmami, leady, nabídkami, zakázkami, projekty, schůzkami, helpdeskem, incidenty, změnami, AI a reportingem.

## Jak začít

1. Nový uživatel začíná dokumentem `00-rychly-start.md`.
2. Každý člen týmu si přečte `01-zakladni-pravidla-prace.md` a `02-role-a-odpovednosti.md`.
3. Obchod čte hlavně dokumenty `03` až `05`.
4. PM a delivery tým čte hlavně dokumenty `05` až `07`.
5. Podpora čte hlavně dokumenty `08` a `09`.
6. Vedení a vlastníci procesů čtou hlavně dokumenty `10` až `12`.

## Základní pravidlo

Chat je na rychlou domluvu. Úkol je na práci, která má vlastníka a termín. Karta je na evidenci dat, vztahů a stavu. Dokument je na pravidla, návody, zápisy a delší text. Důležité rozhodnutí nesmí zůstat jen v chatu.
